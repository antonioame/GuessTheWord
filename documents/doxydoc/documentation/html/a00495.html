◆ getSourceMapWordFrequency()

Map< String, Integer > gruppo05.gtwserver.sourcemanager.internal.analysis.SourceAnalyzer.getSourceMapWordFrequency ( Stream< String > sourceWords)

Elabora uno stream di token puri calcolando la frequenza di ciascuna parola valida.

Parametri
[in]sourceWordsLo stream di token testuali (solo parole) estratti dalla sorgente.
Restituisce
Una mappa contenente le parole normalizzate in minuscolo come chiavi e il rispettivo numero di occorrenze.
Precondizione
Lo stream sourceWords non deve essere nullo e deve contenere stringhe alfanumeriche.
Postcondizione
La mappa restituita contiene solo chiavi in minuscolo e ignora i termini presenti nel set delle stopWords.
