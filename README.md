## Struttura del Progetto
Il progetto è suddiviso nei seguenti moduli e directory principali:

* **[GTWClient](./GTWClient)**: Modulo contenente l'applicazione client.
* **[GTWServer](./GTWServer)**: Modulo contenente l'applicazione server.
* **[GTWShared](./GTWShared)**: Modulo condiviso tra i 2 precedenti, che raccoglie le classi comuni, i modelli dei dati e i protocolli di comunicazione.
* **[eseguibili](./eseguibili)**: Directory contenente le risorse come richieste dalle direttive del progetto.
* **[shellscripts](./shellscripts)**: Script di utilità, consultare il [relativo README](./shellscripts/README.md).
