Descrizione dettagliata

Test unitari per ServerLoginManager.

Poiché ServerLoginManager#validateInfo(String, String) interagisce con componenti JavaFX non disponibili, si utilizza una Testable Subclass, che sovrascrive i metodi di notifica UI, isolando la logica di business e rendendola verificabile in modo deterministico e senza dipendenze grafiche.

I test coprono i seguenti scenari:

  • Credenziali corrette => callback di successo invocato
  • Username inesistente => callback NON invocato, errore notificato
  • Password errata => callback NON invocato, errore notificato
  • Callback non impostato => nessuna NullPointerException
  • Username null => callback NON invocato
