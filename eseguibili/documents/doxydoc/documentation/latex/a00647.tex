\doxysubsection{Descrizione dettagliata}
Oggetto per il trasferimento dati (DTO) flessibile per la comunicazione tra livello di Rete e UI. 


\begin{DoxyItemize}
\item 
\end{DoxyItemize}

Questa classe implementa il pattern architetturale \doxylink{a00659}{Builder}. Garantisce la creazione di oggetti immutabili (read-\/only) le cui variabili vengono inizializzate solo se strettamente necessarie in base al {\ttfamily \doxylink{a00663}{Message\+Type}} ricevuto, prevenendo stati inconsistenti nella UI.
