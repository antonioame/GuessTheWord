\doxysubsection{Descrizione dettagliata}
Test unitari per {\ttfamily \doxylink{a00347}{Server\+Login\+Manager}}. 

Poiché {\ttfamily \doxylink{}{Server\+Login\+Manager\#validate\+Info(\+String, String)}} interagisce con componenti Java\+FX non disponibili, si utilizza una Testable Subclass, che sovrascrive i metodi di notifica UI, isolando la logica di business e rendendola verificabile in modo deterministico e senza dipendenze grafiche.

I test coprono i seguenti scenari\+: 
\begin{DoxyItemize}
\item Credenziali corrette =\texorpdfstring{$>$}{>} callback di successo invocato 
\item Username inesistente =\texorpdfstring{$>$}{>} callback NON invocato, errore notificato 
\item Password errata =\texorpdfstring{$>$}{>} callback NON invocato, errore notificato 
\item Callback non impostato =\texorpdfstring{$>$}{>} nessuna Null\+Pointer\+Exception 
\item Username null =\texorpdfstring{$>$}{>} callback NON invocato 
\end{DoxyItemize}
