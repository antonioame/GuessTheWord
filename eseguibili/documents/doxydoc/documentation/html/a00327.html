Descrizione dettagliata

Factory e Controller di rete di altissimo livello per il lato Client.

Questa classe estende NetworkConnectionCreator per gestire l'instaurazione della connessione TCP con il server. Agisce come mediatore tra il layer di rete e l'interfaccia grafica, ricevendo i messaggi, convertendoli in DTO e aggiornando la UI in modo thread-safe.

◆ handleServerDisconnect()

void gruppo05.gtwclient.networking.ClientConnectionCreator.handleServerDisconnect ( )
private

Gestisce la disconnessione improvvisa del server.

Viene invocata dal thread di rete (non Thread JavaFX). Usa Platform#runLater per mostrare un Alert bloccante sul thread UI e poi eseguire il callback configurato (navigazione per tornare alla LoginView).
