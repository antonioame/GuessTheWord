Esegue una tokenizzazione pura su una riga di testo, estraendo solo parole valide. Poiché l'estetica è gestita altrove, elimina apostrofi e punteggiatura trasformandoli in spazi.

Parametri
[in]lineLa riga testuale da pulire.
Restituisce
Uno Stream contenente i singoli token puliti.
Postcondizione
Parole composte per elisione (es. "dell'animo") vengono divise in due token puri ("dell", "animo").

◆ readRawLines()

Stream< String > gruppo05.gtwserver.sourcemanager.internal.io.IOManager.readRawLines ( Source source) throws SourceNotFoundException

Legge il contenuto grezzo (raw) del file sorgente per preservarne l'estetica. Utilizzato dal generatore di domande per mantenere intatta la formattazione, la punteggiatura e le maiuscole originali dell'autore.

Parametri
[in]sourceL'oggetto sorgente testuale da cui leggere.
Restituisce
Uno Stream di stringhe, dove ogni elemento rappresenta una riga intatta del file originale.
Eccezioni
SourceNotFoundExceptionSe il file non esiste, è inaccessibile o il percorso non è valido.
Postcondizione
Lo stream restituito deve essere chiuso dal chiamante (es. tramite try-with-resources) per liberare il file lock.

Legge il file sorgente estraendone un flusso di sole parole pure. Utilizzato esclusivamente dall'analizzatore per il calcolo statistico delle frequenze, elimina completamente qualsiasi segno di interpunzione.

Uno Stream di token rappresentanti le singole parole, privi di punteggiatura.

Postcondizione
Lo stream restituito non contiene stringhe vuote o segni di punteggiatura isolati.
