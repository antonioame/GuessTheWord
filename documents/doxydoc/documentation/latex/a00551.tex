\doxysubsection{Descrizione dettagliata}
Classe di utility per il testing e il debug del Database. 

Consente il popolamento diretto delle tabelle del database tramite query SQL batch, bypassando i \doxylink{a00387}{DAO} per garantire uno stato iniziale pulito, controllato e deterministico prima dell\textquotesingle{}esecuzione dei test d\textquotesingle{}integrazione o di sistema.

Inizializza il database con un dataset completo di debug senza ricorrere ai \doxylink{a00387}{DAO}. 

Il metodo esegue l\textquotesingle{}intera procedura di popolamento all\textquotesingle{}interno di una transazione SQL atomica. Prima dell\textquotesingle{}esecuzione dei batch, vengono attivati i vincoli di integrità referenziale (Foreign Keys). In caso di errore durante uno qualsiasi degli inserimenti, viene invocato il rollback per ripristinare lo stato precedente.
