Descrizione dettagliata

Oggetto per il trasferimento dati (DTO) flessibile per la comunicazione tra livello di Rete e UI.

Questa classe implementa il pattern architetturale Builder. Garantisce la creazione di oggetti immutabili (read-only) le cui variabili vengono inizializzate solo se strettamente necessarie in base al MessageType ricevuto, prevenendo stati inconsistenti nella UI.
