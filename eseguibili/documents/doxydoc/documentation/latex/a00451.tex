\doxysubsection{Descrizione dettagliata}
Configurazione dei parametri per la generazione delle domande. 

\begin{DoxyInvariant}{Invariante}
Il numero di periodi deve essere maggiore di zero. 

La massima frequenza delle parole deve essere maggiore di zero. 

Il numero massimo di parole simili nel testo della domanda deve essere maggiore o uguale a zero. 

L\textquotesingle{}offset di shifting deve essere maggiore o uguale a zero. 
\end{DoxyInvariant}
